# AgentOrg Overview

AgentOrg lets one person run a company staffed by AI agents.

## Setup (2 minutes)

1. `npx agentorg init` — pick a template, enter API key
2. `npx agentorg start` — company is running
3. Open dashboard at http://localhost:3100 or manage via Telegram

## How It Works

- **Agents** have names, personalities, skills, and memory
- **Org chart** defines who reports to whom — tasks flow down, problems flow up
- **Heartbeats** make agents proactive — they wake on schedule, check for work, act, report
- **Orchestrator** checks every action — permissions, budgets, safety, approvals
- **Skills** let agents interact with the real world — browse, email, write files
- **Skill graph** defines reusable workflows — every blog post follows the same quality pipeline

## Manage Three Ways

- **Telegram**: /status, /approve, /hire — manage from your phone
- **Dashboard**: point-and-click forms, drag-drop org chart
- **YAML**: edit config directly for full control

All three stay in sync automatically.
